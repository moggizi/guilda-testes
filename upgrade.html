⚠️ ATENÇÃO: Por que cobramos por algumas funcionalidades?
Este projeto foi feito exclusivamente para ajudar a comunidade na administração de guildas de FF. Os valores são bem acessíveis e servem apenas para cobrir nossos custos de banco de dados e manter o projeto ativo, não para gerar lucro. Fizemos na intenção de ajudar, mas não conseguimos cobrir todos os custos sozinhos!
